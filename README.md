# Set up front-end

1. Clone the repository

```bash
git clone https://github.com/Calesi19/Rexburg-Arepas.git
cd Rexburg-Arepas
```

2. Install the dependencies

```bash
npm install
```

3. Run the

```bash
npm run dev
```

# Set up back-end

The back-end is an "Appwrite"" project. You can go to the appwrite website and create a project.
